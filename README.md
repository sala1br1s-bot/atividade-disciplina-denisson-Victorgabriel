## 📖 Descrição do Projeto
Este projeto foi desenvolvido em HTML no VS Code com o objetivo de apresentar conceitos básicos de desenvolvimento web, como criação de páginas, uso de tabelas, links, títulos, parágrafos e estilização com CSS. O sistema demonstra a organização de informações em páginas web, incluindo exemplos de navegação entre arquivos e exibição de dados em tabelas. Além disso, foram aplicados conceitos fundamentais de estruturação de páginas e formatação visual, permitindo compreender a construção inicial de sites utilizando HTML e CSS.
